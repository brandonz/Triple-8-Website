# Triple-8-Website
Triple 8 Website

Install Global Packages
npm install -g babel
npm install -g babel-cli

Add Plugins
npm install babel-core
npm install babel-loader
npm install babel-preset-react
npm install babel-preset-es2015

Start Server
npm start

Compile
npm run compile
